Rules
1. alpha,number and _ (underscore)
2. should not start with number
3. Both upper and lower case alphabets are used but not treated as same
4. First letter can be _ ex _name is a valid Variables
5. Variable creation happen when they are assigned with a value. Ex name = "Beniton" ( mean name variable is assigned with value Beniton)
6. Keywords should not be used as variables For example print is a keyword we cannot use print as a variable

Valid variables

A9
_a9

9a
a.A ( period is not allowed)
